# 미세먼지 체감 위험도 계산기

## 프로젝트 소개
PM2.5,PM10,온도,습도,풍속을 입력하면 개인 체감 위험도를 자동으로 계산해주는 프로그램입니다.
단순 미세먼지 수치가 아닌 여러 환경 변수를 종합해 실제 체감 위험도를 더 정확하게 반영하고자 했습니다.

## 개발 동기
미세먼지의 위험도는 단순 PM2.5, PM10 수치만으로 판단하면 안되며, 온도,습도,풍속 등 기상 조건에 따라 실제 체감 위험도가 달라질 수 있다고 생각했습니다.
다양한 환경 변수를 종합해 개인 체감 위험도를 더 세밀하게 반영하는 계산기를 만들고자 했습니다.

## 사용 변수 및 가중치
| 변수 | 가중치 | 선택 이유 |
|---|---|---|
| PM2.5 | 35% | 입자가 작아 폐 깊숙이 침투, 건강에 가장 위험 |
| PM10 | 25% | 호흡기에 직접적인 영향 |
| 습도 | 20% | 높을수록 미세먼지 흡착률 증가 |
| 풍속 | 15% | 낮을수록 미세먼지 정체 |
| 온도 | 5% | 활동량에 간접적 영향 |

## 위험도 기준
| 점수 | 등급 | 행동 요령 |
|---|---|---|
| 80점 이상 | 매우 위험 | 외출 금지 |
| 60점 이상 | 나쁨 | 마스크 착용 |
| 40점 이상 | 보통 | 주의 필요 |
| 40점 미만 | 좋음 | 외출 괜찮아요 |

## 사용 방법
~~~bash
python3 미세먼지계산기.py
~~~
실행하면 에어코리아와 기상펑에서 실시간 데이터를 자동으로 가져와 위험도를 계산합니다.

## 기술스택
- Python 3.14
- 에어코리아와 OpenAPI(PM2.5, PM10 실시간 수집)
- 기상청 ASOS API (온도, 습도, 풍속, 실시간 수집)
