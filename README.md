# 🌫️ 미세먼지 체감 위험도 계산기
실시간 API로 수집한 PM2.5, PM10, 온도, 습도, 풍속을 종합해
**실제 체감 위험도**를 계산하는 웹 애플리케이션입니다.

🌐 **실시간 웹사이트**: https://web-production-2cdde.up.railway.app

---

## 개발 동기
단순 PM2.5, PM10 수치만으로는 실제 체감 위험도를 정확히 반영하기 어렵습니다.
온도, 습도, 풍속 등 기상 조건에 따라 위험도가 달라지며,
계절마다 각 인자의 영향력도 다릅니다.
다양한 환경 변수를 종합해 더 정밀한 위험도를 계산하고자 했습니다.

---

## 계산 방식

### PM2.5 · PM10 — 환경부 CAI 비선형 공식
단순 나누기가 아닌 환경부 통합대기환경지수(CAI) 공식을 적용했습니다.
```
Ip = (IHI - ILO) / (BPHI - BPLO) × (Cp - BPLO) + ILO

Ip   : 계산된 CAI 값
Cp   : 실측 농도
BPLO : 해당 구간 농도 하한
BPHI : 해당 구간 농도 상한
ILO  : 해당 구간 CAI 하한
IHI  : 해당 구간 CAI 상한
```

| 등급 | PM2.5 농도 | CAI 범위 |
|------|-----------|---------|
| 좋음 | 0 ~ 15 µg/m³ | 0 ~ 50 |
| 보통 | 16 ~ 35 µg/m³ | 51 ~ 100 |
| 나쁨 | 36 ~ 75 µg/m³ | 101 ~ 250 |
| 매우 나쁨 | 76 ~ 150 µg/m³ | 251 ~ 500 |

### 계절별 가중치 — 논문 기반
조우싱(2014) *서울시 기상인자와 미세먼지의 통계학적 특성분석* 논문의
계절별 피어슨 상관계수(Table 4.5~4.8)를 기반으로 가중치를 산출했습니다.

| 인자 | 봄 (3~5월) | 여름 (6~8월) | 가을 (9~11월) | 겨울 (12~2월) |
|------|-----------|------------|-------------|-------------|
| PM2.5 | 38% | 38% | 38% | 38% |
| PM10 | 22% | 22% | 22% | 22% |
| 풍속 | 27% | 17% | 13% | 5% |
| 온도 | 8% | 7% | 18% | 17% |
| 습도 | 5% | 16% | 9% | 18% |

> 겨울철 기온·습도는 양의 상관관계(역전층 효과)로 방향이 반전됩니다.
